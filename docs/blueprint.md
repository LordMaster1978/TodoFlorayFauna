# **App Name**: NatureID

## Core Features:

- Image Recognition: Identify plants, animals, and mushrooms from uploaded images using a pre-trained model. The LLM will act as a tool, making use of its image recognition ability to choose when and how to provide its output.
- Species Information: Display detailed information about the identified species, including habitat, characteristics, and potential look-alikes.
- User location: Use the GPS to identify the region, and filter out Flora, Fauna and Mushrooms, that don't appear in the area.
- Favorites List: Allow users to save identified species to a personal favorites list.

## Style Guidelines:

- Primary color: Forest green (#3D5A39) to evoke nature and growth.
- Background color: Light beige (#F5F5DC) for a natural, earthy feel.
- Accent color: Burnt orange (#A0522D) to highlight key elements and calls to action.
- Body and headline font: 'Alegreya', a serif font with an elegant and intellectual feel.
- Use simple, line-based icons representing different types of species (plants, animals, mushrooms).
- Clean, minimalist layout with a focus on large, high-quality images.
- Subtle transitions and animations to enhance user engagement.
